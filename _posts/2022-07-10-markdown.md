## 마크다운 언어란?

마크다운 언어(Markdown language)는 **경량 마크업 언어(Lightweight Markup language)** 이다.

일반적으로 사용되는 마크업 언어는 복잡한 태그 구조를 가질 수 있는 단점이 있었다.  

하지만 마크다운 언어는 태그 구조를 없애고 문법이 쉽고 간단한 것이 특징이다.

마크다운은 2004년 존 그루버(John Grouber)와 아론 스워츠(Aaron Swartz)가 만들었으며 

존 그루버는 마크다운의 디자인 목표로 가능한 읽기 쉽게 만드는 것이라고 한다.

파일의 확장자는 .md 이며 Github에서 작성되는 문서들(예를 들어 README.md)도 마크다운으로 작성된다.

## 문법

#### 헤더
```
# -> H1
## -> H2
### -> H3
#### -> H4
##### -> H5
###### -> H6
```
# H1
## H2
### H3
#### H4
##### H5
###### H6

#### 글씨 모양
```
이텔릭체: *별표(asterisks)* 혹은 _언더바(underscore)_
두껍게: **별표(asterisks)** 혹은 __언더바(underscore)__
취소선: ~~물결표시(tilde)~~
밑줄: `<u></u>`
```
*이텔릭체*    
**두껍게**  
~~취소선~~  
<u>밑줄</u>  

#### 코드 블럭
```
  ``` 코드 ```
```
#### 목록
```
 *, +, - 지원
* 목록 1
* + 목록 2
* + - 목록 3
```
* 목록 1
* + 목록 2
* + - 목록 3

#### 수평선
```
<hr/>
```

#### 링크
```
외부링크: <http://example.com/>
외부링크(링크 숨김) : [Title](link)
이메일링크: <address@example.com>
```
외부링크 예시: <https://kimjinmyeong.github.io/>  
링크를 숨긴 외부링크 예시 : [MyGitBlog](https://kimjinmyeong.github.io/)  
이메일 링크: <kjmdkdlel@gmail.com>
